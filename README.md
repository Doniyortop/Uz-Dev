# UzDev Hub - IT Маркетплейс Узбекистана

Современная платформа для поиска IT-услуг и фрилансеров в Узбекистане. Поддерживает русский и узбекский языки.

## 🚀 Возможности

### 🗄️ Архитектура и База данных
- **Supabase интеграция**: Полноценная база данных PostgreSQL в облаке
- **Реальная авторизация**: Вход через Google, GitHub, Email/пароль
- **Row Level Security**: Безопасный доступ к данным

### 🔍 Поиск и Фильтрация
- **Живой поиск**: Мгновенная фильтрация услуг по названию и навыкам
- **Расширенные фильтры**: Фильтрация по цене, рейтингу, категориям
- **Семантический поиск**: Поиск по описанию и тегам

### ⭐ Доверие и Репутация
- **Система отзывов**: Оценки и текстовые отзывы после выполнения заказа
- **Портфолио**: Галерея работ с возможностью загрузки изображений
- **Статус "В сети"**: Индикатор активности исполнителей
- **Рейтинги**: Автоматический расчет среднего рейтинга

### 📱 UX/UI Оптимизации
- **Скелетоны загрузки**: Красивые индикаторы загрузки
- **Мобильная адаптация**: Mobile-first подход для Узбекистана
- **Анимации**: Плавные переходы и микро-взаимодействия

### 🔍 SEO Оптимизация
- **Динамические meta-теги**: Автоматическая генерация для каждой страницы
- **Мультиязычные URL**: Поддержка русских и узбекских URL
- **Структурированные данные**: OpenGraph и Twitter Card поддержка

## 🛠️ Технологический стек

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Storage**: Supabase Storage
- **Icons**: Lucide React
- **Animations**: Framer Motion

## 📦 Установка и запуск

1. **Клонирование репозитория**
```bash
git clone <repository-url>
cd "IT market uzb"
```

2. **Установка зависимостей**
```bash
npm install
```

3. **Настройка переменных окружения**
Создайте файл `.env.local`:
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

4. **Настройка Supabase**
- Создайте новый проект в [Supabase](https://supabase.com)
- Выполните SQL из файла `lib/supabase/database.sql`
- Настройте OAuth провайдеры (Google, GitHub) в Authentication Settings

5. **Запуск проекта**
```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## 📁 Структура проекта

```
├── app/                    # Next.js App Router
│   ├── [lang]/            # Мультиязычные маршруты
│   │   ├── services/       # Страница услуг
│   │   ├── dashboard/      # Личный кабинет
│   │   └── freelancers/   # Профили фрилансеров
│   └── auth/              # Аутентификация
├── components/            # React компоненты
│   ├── shared/           # Общие компоненты
│   └── ui/               # UI компоненты
├── lib/                  # Утилиты и конфигурации
│   └── supabase/         # Supabase функции
└── types/                # TypeScript типы
```

## 🗄️ Структура базы данных

### Таблицы:
- `profiles` - Профили пользователей
- `services` - Услуги и предложения
- `categories` - Категории услуг
- `reviews` - Отзывы и рейтинги
- `portfolio_items` - Портфолио работ

### Особенности:
- Row Level Security (RLS) для безопасности
- Автоматические временные метки
- Индексы для производительности
- Внешние ключи для целостности данных
